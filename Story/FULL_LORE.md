# 1. COSMOLOGY FOUNDATION

## 1.1 Creation of Andromeda

* The Andromeda galaxy is **artificially created reality**
* Created by:

  * **Xelvelt (God of Light)**
  * **Cyria (Goddess of Darkness)**

### Creation Mechanism:

* Not physics-based → **imagination-based manifestation**
* Reality behaves like:

  ```python
  universe = merge(light.construct(), darkness.shape())
  ```

### Key Concept:

* Light defines structure
* Darkness defines depth/possibility

---

## 1.2 The Split Catastrophe

A critical divine event:

* Xelvelt’s vessel: **Elion Velheart**
* Cyria’s vessel: **Selia Velheart**

### Incident:

* Elion **splits Cyria’s power**
* Results:

| Entity     | Description                                         |
| ---------- | --------------------------------------------------- |
| **Xelisa** | Raw, unfiltered divine darkness (chaotic, dominant) |
| **Selia**  | Gentle remnant (incomplete, passive, hollow)        |

### Consequence:

* Cyria no longer whole
* Reality becomes **unstable at the metaphysical layer**

## 2.3 Capabilities

* Direct god ↔ host dialogue
* Partial or full **possession**
